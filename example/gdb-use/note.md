# gdb 调试工具

## 描述

gdb 在命令行中可以很好的调试 c 语言代码，可以单步执行断点调试，查看变量的值与指针。


## 使用

### 生成可调试程序

使用 gcc 编译时加入 `-g` 参数生成的程序才能够被调试。

```
gcc -g ./main.c
```


### 进入调试

在 gdb 后面跟上执行程序地址。

```
gdb ./main.out
```


### 终止退出

在 gdb 中你按 `ctrl+c` 尝试退出，发现无法终止程序。这时候使用 `q` 命令或按 `ctrl+d` 才能够顺利退出程序。


### gdb 命令

* `list` or `l` 查看代码与行号（部分显示在按回车后会显示更多）
* `start` 开始执行
* `p [变量名]` 打印该变量
* `n` 向下执行
* `s` 进入执行函数
* `bt` 显示函数执行堆栈
* `f [堆栈序号]` 切换到指定执行函数及作用域中
