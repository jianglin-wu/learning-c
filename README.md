## 编译

### 单个源文件编译
在 Linux 下自带了工具 `gcc`，使用它可以将你的 C 代码编译为可以直接运行的二进制文件。

执行这条命令将 `hello.c` 文件编译成二进制，并输出为 `hello.out` 文件。
```bash
$ gcc -o hello.out hello.c
```
